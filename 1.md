# Cloudflare Popular Open Source Toolchains

A curated list of popular and emerging open-source toolchains and tools from Cloudflare, focusing on 2024-2025 developments.

## specialized AI Tools

*   **[Workers AI](https://developers.cloudflare.com/workers-ai/)**: A framework allowing developers to run machine learning models, including LLMs, on Cloudflare's global network using Nvidia GPUs.
*   **[Model Context Protocol (MCP)](https://www.anthropic.com/news/model-context-protocol)**: An open standard for connecting AI agents to external systems and data sources, actively supported by Cloudflare.
*   **[Firewall for AI](https://blog.cloudflare.com/firewall-for-ai/)**: A security layer designed to protect Large Language Models (LLMs) from abuses and attacks.
*   **[AI Labyrinth](https://blog.cloudflare.com/ai-labyrinth/)**: A tool to combat AI data scraping by serving deceptive content to unauthorized bots.

## Development & Infrastructure

*   **[Wrangler](https://github.com/cloudflare/workers-sdk/tree/main/packages/wrangler)**: The official command-line interface (CLI) for building, testing, and deploying Cloudflare Workers and Pages.
*   **[Cap'n Web](https://blog.cloudflare.com/capn-web/)**: A high-performance, open-source RPC (Remote Procedure Call) protocol designed for the modern web (browsers and servers).
*   **[OpenPubkey SSH (OPKSSH)](https://github.com/openpubkey/openpubkey)**: An open-source project integrating single sign-on (SSO) with SSH, simplifying secure access.
*   **[Project Alexandria](https://blog.cloudflare.com/project-alexandria/)**: Cloudflare's initiative to support independent open-source projects like the Ladybird browser and Omarchy Linux.
*   **[Node.js CDN Worker](https://github.com/nodejs/nodejs-dist-indexer)**: Used by the OpenJS Foundation to serve Node.js release assets efficiently.

## Core Ecosystem

*   **[Cloudflare Workers](https://workers.cloudflare.com/)**: The core serverless platform allowing code execution at the network edge.
*   **[WARP](https://one.one.one.one/)**: The high-performance VPN client, with its WireGuard implementation (BoringTun) open-sourced in Rust.
*   **[cdnjs](https://github.com/cdnjs/cdnjs)**: The #1 free and open-source CDN for web libraries, powered and maintained by Cloudflare.
