# Cloudflare Web FrameworkCodes & starters

A showcase of high-quality, edge-native web frameworks and starters optimized for Cloudflare Workers and Pages.

## Core Frameworks

*   **[Hono](https://hono.dev/)**: The standard web framework for the edge. Fast, lightweight, and supports TypeScript out of the box.
*   **[Astro](https://astro.build/)**: A modern web framework for content-driven sites, with first-class support for Cloudflare Pages.
*   **[Remix](https://remix.run/)**: A full-stack web framework that leverages the fetch API and runs natively on the edge.

## Starters & Templates

*   **[react-starter-kit](https://github.com/kriasoft/react-starter-kit)**: An edge-native React template including Vite, Hono, and TanStack Router.
*   **[itty-router](https://github.com/kwhitley/itty-router)**: A tiny, zero-dependency router specifically designed for serverless environments like Workers.
*   **[Flareact](https://github.com/flareact/flareact)**: An edge-rendered React framework inspired by Next.js, built specifically for Workers.
*   **[Next.js SaaS Starter](https://github.com/supasub-app/nextjs-saas-starter)**: A complete SaaS boilerplate with Auth and Stripe, optimized for the Cloudflare environment.

## Visual Builders & CMS

*   **[Webstudio](https://webstudio.is/)**: An open-source visual builder that deploys directly to Cloudflare.
*   **[SonicJs](https://github.com/viclafouch/sonicjs)**: A high-performance headless CMS built entirely on Cloudflare Workers.
*   **[Microfeed](https://github.com/microfeed/microfeed)**: A self-hosted CMS for podcasts and blogs that runs on Workers and R2.
