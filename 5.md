# Cloudflare Workers Discord & Telegram Bot Examples

A comprehensive list of open-source starters and templates for building chat bots on Cloudflare Workers.

## ü§ñ Discord Bots

Building Discord bots on Cloudflare Workers is highly recommended due to zero-cold-start implementation of the Discord Interactions API.

| Name | Description | Link | Status |
| --- | --- | --- | --- |
| **cloudflare-sample-app** | **Official**. Discord's official sample app for Cloudflare Workers. Demonstrates verifying signatures and handling slash commands. | [GitHub](https://github.com/discord/cloudflare-sample-app) | Official |
| **discord-hono** | **Robust**. A lightweight Discord API wrapper built on Hono, capable of running on Cloudflare Workers, Deno, and Bun. | [GitHub](https://github.com/luudis/discord-hono) | Popular |
| **slshx** | A framework-agnostic Discord Interactions handler, perfect for Workers. | [GitHub](https://github.com/slshx/slshx) | Maintained |
| **awwbotcf** | A simple example bot that fetches content from Reddit, originally by Google Devs, adapted for CF Workers. | [GitHub](https://github.com/JustinBeckwith/awwbotcf) | Demo |
| **discord-interactions-worker** | A clean, boilerplate-free starter for Discord Interactions on Workers. | [GitHub](https://github.com/ssMMiles/discord-interactions-worker) | Template |

| Name | Description | Link | Status |
