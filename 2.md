# Cloudflare Popular Open Source Applications

A showcase of high-quality, open-source applications and frameworks built on Cloudflare Workers and Pages.

## Frameworks & Starters

*   **[Hono](https://hono.dev/)**: A small, fast, and ultrafast web framework for the Edges. It's becoming a standard for building APIs on Cloudflare Workers.
*   **[react-starter-kit](https://github.com/kriasoft/react-starter-kit)**: An edge-native React template including Vite, Hono, TanStack Router, and Terraform, optimized for scalability.
*   **[itty-router](https://github.com/kwhitley/itty-router)**: A tiny, zero-dependency router specifically designed for Cloudflare Workers and other serverless environments.
*   **[flareact](https://github.com/flareact/flareact)**: An edge-rendered React framework built for Cloudflare Workers, inspired by Next.js.

## Applications & Utilities

*   **[Webstudio](https://webstudio.is/)**: An open-source visual builder that connects to any headless CMS, capable of deploying directly to Cloudflare.
*   **[Supermemory](https://github.com/supermemoryai/supermemory)**: An AI-powered second brain and memory engine, focusing on speed and scalability using Cloudflare's edge.
*   **[cf-workers-status-page](https://github.com/eidam/cf-workers-status-page)**: A clean, serverless status page that runs entirely on Cloudflare Workers, KV, and CRON Triggers.
*   **[GDIndex](https://github.com/maple3142/GDIndex)**: A Google Drive Index built with Vue, allowing you to list files from your Google Drive on Cloudflare Workers.
*   **[SonicJs](https://github.com/viclafouch/sonicjs)**: A headless CMS built entirely on Cloudflare Workers, focusing on performance.
*   **[Vmail](https://github.com/oiov/vmail)**: A temporary email service deployable to Cloudflare Workers, using D1 for storage.
*   **[Clawdbot](https://github.com/clawdbot/clawdbot)**: An open-source AI agent built on Anthropic's Claude, leveraging Cloudflare's infrastructure for scalability.

## Official Templates

*   **[Cloudflare Workers Templates](https://github.com/cloudflare/templates)**: The official repository containing starter templates for Astro, Next.js, Remix, Vue, and more, covering use cases from SaaS to AI chat apps.
